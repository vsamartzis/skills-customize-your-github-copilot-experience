# 📘 Assignment: Data Structures in Python

## 🎯 Objective

Master Python's core data structures by building a student grading system and analyzing class performance. You'll learn how to effectively use lists, dictionaries, sets, and tuples to organize, manipulate, and analyze real-world data.

## 📝 Tasks

### 🛠️ Task 1: Build a Student Grading System

#### Description
Create a program that manages student grades using multiple data structures. You'll store student information, calculate statistics, and track performance across subjects.

#### Requirements
Completed program should:

- Use a **list** to store a collection of student records
- Use **dictionaries** to represent each student (name, ID, grades for multiple subjects)
- Calculate average grades using the data in dictionaries
- Display formatted grade reports showing student names and their averages


### 🛠️ Task 2: Analyze Class Performance

#### Description
Implement functions to analyze class performance data using sets and dictionaries. Find patterns in student performance and generate summaries.

#### Requirements
Completed program should:

- Use a **set** to identify unique subjects offered
- Use a **set** to find students who passed all subjects (grades >= 70)
- Use a **tuple** to return immutable performance summaries
- Display: total students, subjects offered, passing students, and class average


### 🛠️ Task 3: Generate Performance Insights

#### Description
Create a bonus feature that provides actionable insights by combining all data structures to identify trends and areas for improvement.

#### Requirements
Completed program should:

- Use dictionaries to map subjects to their average grades
- Use a set to identify subjects where most students struggled (average < 70)
- Use a tuple to represent a performance report (subject, average_grade, needs_improvement)
- Display recommendations for which subjects may need extra tutoring or review
